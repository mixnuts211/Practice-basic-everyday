Frontend Mentor를 활용하여 30개의 UI 제작을 연습합니다. 작업 방식은 다음과 같습니다.
  • 해당 챌린지에서 1순위로 가져가는 부분은 실무와 같은 빠른 코드 작성입니다.
  • 피그마나 스케치 파일 없이 진행되므로 섬세한 px 단위의 작업은 후순위로 두거나 생략합니다.
  • 작업 시간은 VS Code extension `Waka time`을 기준으로 기록합니다.
  • UI단에서 작업 가능한 웹표준, 접근성 등을 지키고 고려하여 코드를 작성합니다.
  • 해당 UI 작업들은 Chrome 브라우저를 기준으로 작성되었습니다.
